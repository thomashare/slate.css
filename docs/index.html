key terms: SS=screen size, X=number range

Slate is the easy to work css framework for building responsive, gluten-free, developer friendly websites.

The Basics

columns:

xs=extra-small, sm=small, md=medium, lg=large and xl=extra-large

By default, columns range from 0-12. 0 meaning hidden, 12 meaning full width. Each column level is equal to 100% of screen width, divided by the 'max-columns' setting. By default 'max-columns' is set to 12. Ex: 'xs-4 sm-8 md-12'.

By giving the 'xs' screen size a level of 0, the element on that specific screen size will become hidden. Alternatively, you can use 'xs-hide' or 'md-hide' etc. In addition, each column level will increase by 100% / 'max-columns'.

Nav Menus:

sm=small nav menu, md=medium, lg=large

ex: <nav class="sm">

Use by adding the 'sm' 'md' 'lg' or 'xl' class to your nav element. These classes help define the importance of a nav.

Fonts:

font-size: format 'font-size-X'

ex: <div class="font-size-5">

'X' represents a level amount. Each level of font size is incremented by 0.25rem and levels go up to 25.

Font sizes help define importance in textual elements. Use 'font-size-2' to give a smaller font size or 'font-size-7' to give a larger font size to an element.

Text Effects:

  • text-glow: Adds a light text shadow to text elements in order to help better distinguish dark text on a slightly dark or grey background.
  • text-shadow: Adds a dark text shadow to text elements in order to help better distinguish light text on a lighter background.

Margins: in format 'SS-margin-X'

ex: <div class="xs-margin-2">

'X' represents a level amount. Each level of margin is incremented by 5px and levels go up to 25.

Margins help space out your content. Use 'xs-margin-2' to give a margin level of two all around the object. Use 'xs-margin-2' to give a margin level of two on the top and bottom of the object. Take it a step further and add an extra -3 to that class to give a margin level of three on the left and right of the object (Ex: 'xs-margin-2-3').

Specify top, bottom, left or right margin with class (Ex: 'xs-margin-top-3').

Don't want any margin? Use class (Ex: 'xs-no-margin'). *Useful when a full-screen navigation menu is desired.

Margins range from 1-20.

Padding: in format 'SS-pad-X'

ex: <div class="xs-pad-2">

As with margins, this class adds a padding to an object in a similar fashion of 'SS-pad-X-X'. Padding levels go up to 25.

Specify top, bottom, left or right padding with class (Ex: 'xs-pad-top-3').

Don't want any padding? Use class (Ex: 'xs-no-padding').
