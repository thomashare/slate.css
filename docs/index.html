Slate.css is an unopinionated CSS framework for rapid web layout and design.

Introduction

Now, instead of using floats and endless helper functions, Slate uses the preexisting grid architecture for layouts. This helps minimize the code necessary and improves load times.

Layout

In modern CSS frameworks, you create layouts using helper classes such as “xs-4” or “md4” or even “sm-col-6”. With Slate, these classes are no longer needed.

Slate was created to help you break away from that habit and begin relying on the grid system built-in to modern browsers.

Slate uses a 12-column span system. Use “s1” through “s12” to represent your column span size.

Render

Add the “render” class to any grid element to create a render.

The render class helps out with visually laying out your content. This class is strictly meant for mockups and visual representation.

All elements inside the render class will be given a default height of 40px. Add a class of “sm” to make the element's height 100px, “md” to make it 250px, and “lg” to make it 500px.

Debug

Add the class “debug” to any element to outline it in red. This is useful when working with margins/padding and alignment.

If you have an element with no background and/or no border, giving it a red outline will help you clearly see the content box.

Forms

Forms can use simple input, textarea, and buttons without wrappers.

Place form inputs inside a form element or a div element with a “grid” class to control input widths with span sizes.

Or, just wrap your form elements in a form tag to auto fit your inputs to the grid.

Responsive

Slate will automatically expand all spans to 100% on screens smaller than 767px.

This means any sidebars will displayed before or after content. To avoid this, add a “mobile-off” class to the element. Notice the div with the “mobile-off” is hidden on smaller devices.

Similarly, adding the “mobile-on” class to an element will only show it on smaller devices.

Content

Writing content is easier than ever. When writing content, just use normal html5 tags.

Text Alignment

Center text with the “txt-center” class and align-right with the “txt-right” class.

Media

Any element can use span tags, even images and embeds.

Due to limitations in iframes, iframes must be wrapped in an “iframe” class element.

Center elements by adding a “center” class to the container element.

Align elements to the bottom by adding a “bottom” class to the container element.
